Yeoman

YEOMAN GENERATORS Community-based driven workflow autogenerator for different framework. Yeoman provides an optimized scaffolding and workflow experience for creating compelling web applications. Developers use Yeoman together with Grunt, for building their projects and Bower for package management.

A typical workflow between this trinity of tools might look like: yo webapp yo angular bower install angular-directives grunt http://yeoman.io/generators.html#snippets http://yeoman.io/faq.html
